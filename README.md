# Drop-Token

This service uses the python Flask framework.

To install Flask:
```
$ pip install Flask
```

To run the service:
```
$ python run.py
```

To run the tests:
```
$ python -m src.test.test_drop_token_game
$ python -m src.test.test_game_manager
```
Note: Tests not exhaustive, wrote them as I needed them.
